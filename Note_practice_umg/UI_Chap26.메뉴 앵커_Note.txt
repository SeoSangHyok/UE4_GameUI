1. 메뉴엥커는 컨텍스트 메뉴를 호출할수 있는 엥커 위젯.

2. 배치후 컨텍스트 메뉴 세팅을 해준후에 이벤트 발생시 Open함수를 이용해서 컨텍스트 메뉴를 연다.

3. ToggleOpen함수를 이용하면 메뉴 토글처리가 가능하다.

4. 오픈된 컨텍스트 메뉴는 마우스인풋을 지원하며 다른곳을 클릭하면 자동으로 UI상에서 제거까지 된다.

5. 다만 오픈된 컨텍스트 메뉴를 클릭하면 지워지지 않는데 이는 수동으로 컨텍스트 메뉴항목 인터랙션 발생시 Close함수를 이용해서 직접 지워줘야한다.

6. 위작업을 하려면 컨텍스트 메뉴가 메뉴엥커 위젯의 레퍼런스를 가지고 있어야하는데 이는 메뉴엥커 위젯의 OnGetMenuContent함수를 오버라이딩해서 처리가능.
   해당 함수는 컨텍스트 메뉴를 직접 생성해주는 함수인데 사용할 컨텐스트 메뉴를 생성한후 해당 메뉴에 메뉴엥커 위젯을 넘기면 된다.
